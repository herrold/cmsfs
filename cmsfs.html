CMS Filesystem Driver and Utility

This page presents the CMS Filesystem Driver and Utility package.
The package can be built as a filesystem driver on Linux or as a utility on any Unix (including Linux and OpenVM).

Pre-built utility executables are available here for

  • Linux for INTeL Pentium (compressed TAR)
  • Linux for S/390 (31-bit) (compressed TAR)
  • Linux for zSeries (64-bit) (compressed TAR)
  • Linux for DEC Alpha (compressed TAR)
  • Solaris for SPARC (compressed TAR)
  • AIX for PowerPC (compressed TAR)
  • HP-UX for PA RISC (compressed TAR)
  • Tru64, OSF/1 for DEC Alpha (compressed TAR)
  • VM/CMS (OpenExtensions) (compressed TAR)

Source code is available as

  • cmsfs.tar, or
  • cmsfs.tar.gz (compressed)

RPMs are available as

  • cmsfs-1.1-0.s390.rpm for mainframe Linux
  • cmsfs-1.1-0.i386.rpm for PC Linux
  • cmsfs-1.1-0.alpha.rpm for DEC Alpha Linux

The official source depot is

ftp://ftp.bmc.com/pub/cmsfs/

and is mirrored at

http://www.linuxvm.org/Patches/
